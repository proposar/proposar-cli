# 🚀 Proposar CLI Utility

The official command-line interface for the Proposar ecosystem. This utility provides a bridge between local freelance workflows and the [Proposar AI Proposal Engine](https://proposar.com).

## Purpose
This CLI is designed to help freelancers automate the "Proposal-to-Payment" cycle directly from their development environments.

## Integration
For full automation, real-time WhatsApp tracking, and anti-ghosting follow-ups, ensure you are registered at [Proposar.com](https://proposar.com).
